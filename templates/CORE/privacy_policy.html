Politique de confidentialité

Dernière mise à jour : 28/10/2025

EasyInvoice s'engage à respecter votre vie privée. Cette page explique nos pratiques en matière de collecte, d'utilisation et de protection de vos données personnelles.

Engagement de confidentialité

EasyInvoice fonctionne entièrement sur votre machine ou serveur local. Aucune donnée n'est transmise à des tiers.

Aucune base de données

EasyInvoice n'utilise aucune base de données. Les informations que vous saisissez dans le formulaire restent dans votre navigateur jusqu'au moment où vous générez le PDF.

Aucun cookie

EasyInvoice ne dépose aucun cookie sur votre ordinateur. Nous n'utilisons aucune technologie de suivi.

Aucun enregistrement de données

Les informations que vous saisissez (nom, adresse, montants, etc.) ne sont jamais stockées, que ce soit dans une base de données ou sur nos serveurs.

Fonctionnement technique

EasyInvoice est une application Python (Django) hébergée sur PythonAnywhere. Voici comment ça fonctionne :

  • Formulaire HTML : Les données sont saisies dans votre navigateur.
  • Traitement Python : Le formulaire est soumis et traité par un script Python sur le serveur PythonAnywhere.
  • Génération PDF : Un PDF est généré instantanément et téléchargé sur votre ordinateur.
  • Suppression automatique : Les données du formulaire sont effacées après génération du PDF.

Format PDF

Les fichiers PDF générés contiennent uniquement les informations que vous avez volontairement saisies dans le formulaire. Une fois téléchargés, ces fichiers sont sous votre contrôle exclusif.

Hébergement PythonAnywhere

EasyInvoice est hébergé sur PythonAnywhere (pythonanywhere.com). Voici les garanties de confidentialité :

  • Les données transitent uniquement par le protocole HTTPS (chiffrement SSL/TLS).
  • Les données ne sont jamais stockées sur le serveur PythonAnywhere.
  • Chaque génération de facture est indépendante et isolée des autres.
  • Les logs du serveur ne conservent aucune trace des données saisies (seulement des traces techniques anonymes).
  • PythonAnywhere n'a pas accès aux données de votre facture car elles sont traitées uniquement en mémoire.

Conformité RGPD

Conformément au Règlement Général sur la Protection des Données (RGPD) :

  • EasyInvoice ne collecte, n'utilise, ni ne transfère vos données à des tiers.
  • Vos données ne sont pas stockées après la génération du PDF.
  • Vous avez un contrôle total sur vos factures générées (elles sont téléchargées localement).
  • Aucune déclaration à la CNIL n'est requise car aucune donnée personnelle n'est conservée.
